## What is LoFi Engine?

LoFi Engine is a desktop application that generates procedural LoFi music using Tone.js, providing a dynamic and unique listening experience every time. Built with Svelte, TypeScript, and Tauri, it offers a lightweight, cross-platform solution for creating your own LoFi studio.

<Info>
LoFi Engine works completely offline. Your experience is fully private, with no internet connection required.
</Info>

## Cross-platform support

LoFi Engine runs on all major desktop platforms:

- **Linux**: Native desktop application
- **macOS**: Native desktop application with private API support
- **Windows**: Native desktop application
- **Web**: Browser-based version at [lofi-engine.vercel.app](https://lofi-engine.vercel.app/)

<Tip>
The desktop app provides the best performance and offline support. The web version is perfect for trying LoFi Engine without installation.
</Tip>

## Core philosophy

LoFi Engine's main focus is:

1. **Customization**: Full control over music, ambience, and visual appearance
2. **Accessibility**: Keyboard shortcuts for every action, multilingual support
3. **Artistic freedom**: Be the artist and create your unique LoFi studio

## Immersion system (Auto DJ)

The Immersion system automatically manages your soundscape with four modes:

- **Music**: Focuses on beat and chords without atmospheric effects
- **Atmosphere**: Adds core weather and nature effects for fuller sound
- **World**: Incorporates specific textures like city sounds or wind for deep immersion
- **Manual**: Gives you full control to mix and match sounds as you please
